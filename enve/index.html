Enve Apartments

Designing a website for apartments

Overview

I worked to design a desktop and mobile version for Enve Apartment in Eastlake. The client wanted a similar flow to the Ballard57 web site, and specifically wanted the layout to be a single page with scrolling and anchor navigation. The idea is to make the site easy to navigate and organized to make apartment hunting a little easier.

Tools Used

Paper and Pen

Key Learnings

Taking the time to find users to test the website and mobile version more to see if they drop off using the single page scrolling through analytics.

If I had more time to work on this project, I would have wanted to interview people ages 21-30 years old to understand more about how they look for apartments.
